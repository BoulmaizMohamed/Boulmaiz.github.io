## üìù Customization Guide

### Update Personal Information
1. Open `index.html`
2. Search for your name and replace with yours
3. Update email, phone, location
4. Replace social media links
5. Update project information

### Change Colors
1. Open `style.css`
2. Modify CSS variables at the top:
```css
:root {
    --primary: #6366f1;
    --secondary: #ec4899;
    --accent: #14b8a6;
    /* ... more colors */
}
```

### Add/Remove Sections
- Each section is clearly marked in `index.html`
- Copy/paste section structure to add new ones
- Update navigation links accordingly

### Update Images
- Replace `profile.png` with your photo
- Update project images (tuba.png, ejabi.png, etc.)
- Maintain aspect ratios for best results
